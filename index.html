Frequently Asked Questions

What exactly does the SOP Generator do?
It's a Python CLI tool that generates professional Standard Operating Procedures for AI automation agencies. You choose a framework (onboarding, sales, implementation, etc.), an industry, a service line, and an agency type, and it produces a complete SOP document with step-by-step instructions, role assignments, time estimates, and quality metrics in Markdown, HTML, and JSON formats.
Do I need to know how to code?
No. The tool has an interactive mode that walks you through every option. Just run `python3 ai_sop_generator.py --interactive` and follow the prompts. If you want to use the command-line mode, it's as simple as `--sop onboarding --industry "Real Estate"`.
Can I customize the generated SOPs?
Absolutely. The Markdown and HTML outputs are designed to be edited. You can also modify the Python source to add custom step descriptions, change templates, or add new industries. The JSON output is perfect for pipeline integration with your own documentation systems.
What's the difference between Standalone, Bundle, and Agency License?
Standalone ($14) gives you just the SOP Generator. Bundle ($37) includes all 15+ products in the AI Revenue Toolkit — prompt packs, playbooks, CLI tools, everything. Agency License ($97) adds commercial use rights, white-label documentation, and the ability to resell to your clients.
What formats does it output?
Three formats: Markdown (for Notion, GitHub wikis, or docs), HTML (styled, ready to share with clients), and JSON (for API integration or custom pipelines). You can choose one format or generate all three at once.
How is this different from the Client Acquisition Playbook?
The Client Acquisition Playbook is a 5-module strategic guide (8,500 words) about how to build your agency's acquisition engine. The SOP Generator is a tool that generates executable SOP documents — step-by-step procedures your team follows. One is strategy, the other is execution. They complement each other perfectly.
